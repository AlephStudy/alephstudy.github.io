Aleph

An Alternative

Gone are the days of paywalled study software, we plan on making a completely free, open-source study tool.

Our Plan

After months of research, we decided Aleph will be an app based tool, not a website.

This is to keep the program as free as possible, since a website based solution would inevitably require some form of host service.

How to build a Study Tool:

Aleph is currently going to be written in:

  • Dart with the Flutter framework

This is so that we can export the code to multiple OS(s) and possibly to a web version in the future.

The Website (what you're looking at):

This website is coded in pure HTML and CSS (and probably JavaScript in the future).
